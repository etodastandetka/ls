# Инструкция по использованию бота-модератора

## Основные функции

1. **Автоматическое удаление сообщений** с запрещенными словами
2. **Мут нарушителей** на 5 минут
3. **Управление списком запрещенных слов** через личные сообщения (только для администратора)

## Команды в личных сообщениях

### Для всех пользователей:
- `/start` - информация о боте
- `/help` - справка
- `/test` - показать список запрещенных слов
- `/check` - проверить статус конфигурации

### Только для администратора (ID: 8203434235):
- `/words` - показать полный список запрещенных слов
- `/add_word` - добавить новое запрещенное слово
- `/remove_word` - удалить слово из списка
- `/cancel` - отменить текущую операцию

## Команды в группах

### Для всех пользователей:
- `/test` - проверить работу бота и показать примеры запрещенных слов

### Только для администратора (ID: 8203434235):
- `/unmute` - размутить пользователя
  - Использование с реплаем: ответьте на сообщение пользователя командой `/unmute`
  - Использование с username: `/unmute @username` (если пользователь упомянут в сообщении)
  - Использование с ID: `/unmute 123456789`

## Как добавить запрещенное слово

1. Напишите боту в личные сообщения
2. Отправьте команду `/add_word`
3. Бот попросит отправить слово
4. Отправьте слово, которое нужно запретить
5. Слово будет добавлено в список

## Как удалить запрещенное слово

1. Напишите боту в личные сообщения
2. Отправьте команду `/remove_word`
3. Бот попросит отправить слово
4. Отправьте слово, которое нужно удалить
5. Слово будет удалено из списка

## Что происходит при нарушении

1. Сообщение с запрещенным словом **удаляется**
2. Пользователь **мутится на 5 минут**
3. Отправляется предупреждение (которое удаляется через 10 секунд)

## Требования для работы в группе

- Бот должен быть **администратором** группы
- Бот должен иметь право **удалять сообщения**
- Бот должен иметь право **ограничивать пользователей** (для мута)

## Файл со словами

Список запрещенных слов сохраняется в файле `forbidden_words.json` в папке бота.
