# Бот-модератор для Telegram

Бот для автоматического удаления сообщений с запрещенными словами в группах Telegram.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` или установите переменную окружения:
```bash
export MODERATOR_BOT_TOKEN="ваш_токен_бота"
```

Или создайте файл `.env` в директории `bot_moderator`:
```
MODERATOR_BOT_TOKEN=ваш_токен_бота
```

3. Настройте список запрещенных слов в файле `config.py`:
```python
FORBIDDEN_WORDS = [
    "спам",
    "реклама",
    # добавьте свои слова
]
```

## Запуск

```bash
python bot.py
```

## Настройка

### Добавление бота в группу

1. Добавьте бота в группу через поиск по username или пригласите по ссылке
2. Сделайте бота администратором группы
3. Предоставьте боту право на удаление сообщений

### Настройка в config.py

- `FORBIDDEN_WORDS` - список запрещенных слов
- `SKIP_ADMINS` - пропускать ли сообщения от администраторов (True/False)
- `SEND_WARNING` - отправлять ли предупреждение при удалении (True/False)
- `WARNING_MESSAGE` - текст предупреждения
- `WARNING_DELETE_SECONDS` - через сколько секунд удалять предупреждение (0 = не удалять)

## Как это работает

1. Бот отслеживает все сообщения в группах, где он является администратором
2. Проверяет текст сообщения на наличие запрещенных слов
3. Если найдено запрещенное слово, удаляет сообщение
4. Опционально отправляет предупреждение пользователю

## Команды

- `/start` - информация о боте (в личных сообщениях)
- `/help` - справка по использованию (в личных сообщениях)

## Требования

- Python 3.8+
- aiogram 3.4.1+
- Права администратора в группе с возможностью удаления сообщений
